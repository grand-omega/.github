# Grand Ωmega

Focused, single-purpose products spanning local AI, sport science, and
specialty e-commerce.

## Projects

- **Project ZUN** — a local image-editing stack on FLUX.1 Fill and FLUX 2
  klein. Inference and LoRA training on a single consumer GPU, with a Rust
  API server and an Android client (iOS soon).
- **Project Maître** — AI bout analysis for competitive fencers and their
  coaches. Server foundations shipped; analysis pipeline in active
  development.
- **Project Yuri** — the Grand Ωmega authorized-dealer storefront for
  **Yuri Workshop**, an Italian luxury fencing-foil maker. In development.
- **Project AGN** *(Agent)* — an agentic framework for academic research:
  paper-writing pipelines, GPU-shader R&D, and an AI conference reviewer
  (coming soon).
